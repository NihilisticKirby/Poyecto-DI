Resumen ejecutivo

La propuesta consiste en una página web con el propósito de presentar información de una manera clara y ordenada, pretendiendoque sea fácil para el usuario interactuar con ésta y tener una experiencia placentera.



Desarrollo

La página cuenta con 6 diferentes páginas de navegación de HTML5: una página de inicio donde se incluye un resumen complto del contenido que será presentosdo, contando con 5 fotografías en esta con navegación a una página con una galería con más de 50 fotografías. Contamos con la información presentada de una manera interactiva para el usuario, con un diseño agradable a la vista y que evita que tengan la sensación de estar leyendo un libro. Contiene una presentación del prototipo de una manera llamativa para que el usuario pueda conocer más acerca del proyecto y dar una retroalimentación al respecto de éste, proporcionando mejoras continuas para asegurar el éxito una vez que el producto sea lanzado.
